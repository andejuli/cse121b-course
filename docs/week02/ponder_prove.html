JavaScript and HTML

In the Tasks for this course, we will manipulate content in HTML files from JavaScript. In order to link a JavaScript file to an HTML file, an HTML <script> tag is used.

The <script> tag is commonly placed before the closing </body> tag as follows: 

<main>
	<section>
		<div>
			<label for="name">Name:</label>
			<span id="name"></span>
		</div>
		<div>
			<label for="food">Favorite Foods:</label>
			<span id="food"></span>
		</div>
	</section>
</main>
<footer>
	&copy;<span id="year"></span> | Biography | Lesson 2
</footer>
<script src="scripts/task2.js"></script>

The "src" attribute of the <script> tag should point to the folder and file name of the JavaScript file relative to the current HTML file.

Accessing HTML elements from JavaScript

Even though this course does not dive deep into HTML, you will need an understanding of how to access HTML elements from JavaScript files.

There are several ways to accomplish this.

  • One is by using the built-in method: document.getElementById('elementId');

    The "elementId" is the value of the "id" HTML attribute. For example, with the following HTML element, <section id="discussion"> the "id" would be "discussion"

    In this case, we would use document.getElementById('discussion'); to get access to this element from JavaScript.

  • Another approach is to use the built-in method: document.querySelector('#elementId')

    The "elementId" is again the value of the "id" HTML attribute. Notice that this approach also requires a pound sign (#) before the name of the "id" value.

    Using the same HTML element above, the JavaScript method would then look like this: document.querySelector('#discussion');

Once we get access to the HTML element, we can change its content by using one of two properties.

  • The first way is to use the "innerHTML" property.

    This would look like the following:

    document.querySelector('#discussion').innerHTML = 'The content we would like to display in the HTML file.';

  • A second way is to use the "textContent" property.

    This would look like the following:

    document.querySelector('#discussion').textContent = 'The content we would like to dsiplay in the HTML file.';

We can also set the attributes of HTML elements by using the built-in method: setAttribute('attributeName', 'attributeValue')

  • For example, we can add a "src" attribute to an "img" element:

    document.querySelector('#mainImage').setAttribute('src', 'images/main.png')
