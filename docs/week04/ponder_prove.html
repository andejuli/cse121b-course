Object Literals

In previous Tasks, we have declared and used individual variables. In this Task, we will learn how to declare objects that can hold multiple name/value pairs as properties.

We will revisit the Biography Task from Lesson 2, this time using object literals.

Data

Complete the following steps

In the task4.js file:

  1. Declare a new variable to hold information about yourself and assign an empty object to the variable (hint: {} )
  2. Inside of the object, add a property named name with a value of your name as a string
  3. Add another property named photo with a value of the image path and name (used in Task 2) as a string
  4. Add another property named favoriteFoods with a value of an array of your favorite foods as strings ( hint: [] )
  5. Add another property named hobbies with a value of an array of your hobbies as strings
  6. Add another property named placesLived with a value of an empty array
  7. Inside of the empty array above, add a new object with two properties: place and length
  8. For each property, add appropriate values as strings
  9. Add additional objects with the same properties for each place you've lived

Output

Complete the following steps

In the task4.js file:

  1. Assign the value of the name property (of the object declared above) to the HTML <span> element with an ID of name
  2. Assign the value of the photo property as the src attribute of the HTML <img> element with an ID of photo
  3. Assign the value of the name property as the alt attribute of the HTML <img> element with an ID of photo
  4. For each favorite food in the favoriteFoods property, create an HTML <li> element and place its value in the <li> element
  5. Append the <li> elements created above as children of the HTML <ul> element with an ID of favorite-foods
  6. Repeat Step 4 for each hobby in the hobbies property
  7. Repeat Step 5 using the HTML <ul> element with an ID of hobbies
  8. For each object in the placesLived property:
    - Create an HTML <dt> element and put its place property in the <dt> element
    - Create an HTML <dd> element and put its length property in the <dd> element
  9. Append the HTML <dt> and <dd> elements created above to the HTML <dl> element with an ID of places-lived
